# ことば落とし（Kotoba Otoshi）

千原ジュニア氏考案の人気ボードゲーム「ことば落とし」をオンラインで楽しめるWebアプリです。
日常会話の中に、指定されたワードを「いかに自然に紛れ込ませるか」を競う、トークと心理戦のゲームです。

## 📝 ゲーム概要

プレイヤーの一人が「落とし役（親）」となり、自分だけが知っている**キーワード（秘密のワード）**を、制限時間内のトーク中に自然に発言することを目指します。
他のプレイヤーは、「どの言葉がキーワードだったのか」を見破ってください。

## 🎮 基本ルール

1. **配役とお題の決定**
   - 2人のうちどちらが「落とし役（親）」か選んでください。親は会話の制限時間を1分,2分,3分の中から選んで「ルームを作成」ボタンを押します。
   - 「落とし役（親）」にだけ、そのゲームで発言すべき「キーワード（秘密のワード）」が表示されます。
   - 子に「ルームID」を教えてあげて下さい。
   - 子は親からもらったルームIDで「参加」しましょう。
   - 子がルームに参加すると「子が参加しました」と親に表示されます。
   - 準備が出来たら「ゲーム開始」ボタンを押してください。
2. **トーク開始**
   - 2人で共通のトークテーマ【お題】に沿って会話を始めます。
3. **ことばを落とす**
   - 「落とし役（親）」は制限時間内に、バレないように先ほどの「キーワード（秘密のワード）」を会話の中に組み込みます。
4. **判定**
   - 制限時間終了後、子は「今のがキーワードだ！」と思う言葉を入力し送信してください。
   - 子がキーワードを当てて正解なら「子の勝ち」
   - 子がキーワードを当てられなくて不正解なら「親の勝ち」となります。

### 勝利条件
- **親の勝利**：子にバレずにお題を言うことができた場合。
- **子の勝利**：お題を正確に見破ることができた場合。
